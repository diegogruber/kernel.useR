Más NLP

2017/07/05

Después del subidón de ayer con el tutorial de cleanNLP, hoy he ido a una charla sobre el package tidytext. El concepto es similar y la presentación fue también muy buena. El package viene acompañado de un libro que tiene buena pinta. Teniendo en cuenta que es algo injusto comparar una charla de 20 minutos con un tutorial de tres horas, mi primera impresión es que cleanNLP le lleva bastante ventaja en cuanto a funcionalidades avanzadas.

He visto un par de charlas más sobre NLP, ambas menos memorables.
